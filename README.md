# AgentSearchBench Website

This is the project website for [AgentSearchBench](https://bingo-w.github.io/AgentSearchBench/), a large-scale benchmark for AI agent search in the wild.

For the code, data, and full documentation, see the [main repository](https://github.com/Bingo-W/AgentSearchBench).
